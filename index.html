លទ្ធផលប្រឡងទទួលបាន

ពិន្ទុ៖ 0 / ១០០

លំហាត់ ១. គណនាលីមីតរាងមិនកំណត់ $\frac{0}{0}$៖ $\lim_{x \to 2} \frac{x^2 - 5x + 6}{x - 2}$ (៥ ពិន្ទុ)
លំហាត់ ២. គណនាលីមីតរាងអនន្តលើអនន្ត $\frac{\infty}{\infty}$៖ $\lim_{x \to \infty} \frac{3x^2 - 4x + 1}{2x^2 + 5}$ (៥ ពិន្ទុ)
លំហាត់ ៣. គណនាលីមីតត្រីកោណមាត្រ៖ $\lim_{x \to 0} \frac{\sin 5x}{2x}$ (៥ ពិន្ទុ)
លំហាត់ ៤. គណនាលីមីតទាក់ទងនឹងចំនួន $e$៖ $\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^{3x}$ (៥ ពិន្ទុ)
លំហាត់ ៥. ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} \frac{x^2 - 1}{x - 1} & \text{បើ } x \neq 1 \\ k & \text{បើ } x = 1 \end{cases}$ ជាប់ត្រង់ $x_0 = 1$ តើ $k$ ត្រូវមានតម្លៃស្មើប៉ុន្មាន? (៥ ពិន្ទុ)
លំហាត់ ៦. រកតម្លៃ $a$ ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} ax + 3 & \text{បើ } x \le 2 \\ 2x - 1 & \text{បើ } x > 2 \end{cases}$ ជាប់លើ $\mathbb{R}$។ (៥ ពិន្ទុ)
លំហាត់ ៧. គេឱ្យអនុគមន៍ $f(x) = \frac{x+2}{x^2 - 4x + 3}$។ តើអនុគមន៍នេះដាច់ (មិនជាប់) ត្រង់តម្លៃ $x$ ណាខ្លះ? (៥ ពិន្ទុ)
លំហាត់ ៨. គណនាដេរីវេនៃអនុគមន៍ $f(x) = (3x^2 - 5x + 2)^4$ (៥ ពិន្ទុ)
លំហាត់ ៩. រករូបមន្តដេរីវេនៃអនុគមន៍ផលចែក $f(x) = \frac{2x + 1}{x - 3}$ (៥ ពិន្ទុ)
លំហាត់ ១០. គណនាដេរីវេនៃអនុគមន៍ត្រីកោណមាត្រ $f(x) = \sin(2x) + \cos^2 x$ (៥ ពិន្ទុ)
លំហាត់ ១១. គណនាដេរីវេទីពីរ $f''(x)$ នៃអនុគមន៍ $f(x) = x \cdot \ln x$ (៥ ពិន្ទុ)
លំហាត់ ១២. រកមេគុណប្រាប់ទិសនៃបន្ទាត់ប៉ះនឹងក្រាបតាងអនុគមន៍ $f(x) = e^{2x}$ ត្រង់ចំណុចដែលមានអាប់ស៊ីស $x_0 = 0$។ (៥ ពិន្ទុ)
លំហាត់ ១៣. ក្នុងប្រអប់មួយមានឃ្លីក្រហម ៤ និងឃ្លីខៀវ ៦។ គេចាប់យកឃ្លី ១ ដោយចៃដន្យ។ រកប្រឡាបដើម្បីចាប់បានឃ្លីក្រហម។ (៥ ពិន្ទុ)
លំហាត់ ១៤. បោះគ្រាប់ឡុកឡាក់មួយគ្រាប់ដែលល្អស្មើជម្រុងចំនួន ១ ដង។ រកប្រូបាបដែលលោតចេញលេខជាចំនួនបឋម។ (៥ ពិន្ទុ)
លំហាត់ ១៥. គេបោះកាក់មួយដ៏ល្អចំនួន ៣ ដង។ រកប្រូបាបដើម្បីឱ្យបោះបានមុខ "រូប" យ៉ាងតិចចំនួន ១ ដង។ (៥ ពិន្ទុ)
លំហាត់ ១៦. គេឱ្យព្រឹត្តិការណ៍ពីរ $A$ និង $B$ ជាព្រឹត្តិការណ៍មិនទាក់ទងគ្នា ដែលមាន $P(A) = 0.3$ និង $P(B) = 0.4$។ គណនា $P(A \cap B)$។ (៥ ពិន្ទុ)
លំហាត់ ១៧. រកសមីការអាស៊ីមតូតឈរនៃក្រាបតាងអនុគមន៍ $y = \frac{3x - 1}{x - 2}$ (៥ ពិន្ទុ)
លំហាត់ ១៨. រកសមីការអាស៊ីមតូតដេកនៃក្រាបតាងអនុគមន៍ $y = \frac{4x^2 + 1}{2x^2 - 3}$ (៥ ពិន្ទុ)
លំហាត់ ១៩. រកកូអរដោនេចំណុចរបត់ (Inflection point) នៃក្រាបតាងអនុគមន៍ $y = x^3 - 3x^2 + 5$ (៥ ពិន្ទុ)
លំហាត់ ២០. រកបរមាតម្លៃនៃអនុគមន៍ $y = -x^2 + 4x + 1$។ តើអនុគមន៍នេះមានតម្លៃអតិបរមា ឬអប្បបរមាស្មើប៉ុន្មាន? (៥ ពិន្ទុ)
